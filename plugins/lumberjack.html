Lumberjack Plugin

Lumberjack Plugin is a plugin for broke protocol that allows you to add the job of lumberjack, without processors just approach the tree with an ax and press "Chop"

Download Here

Configuration:

Map actions: create a lumberjack zone and add TreeWoodcutter in the area, thats trees allow the player to chop.

Config file: The configuration file is located at /Plugins/Lumberjack Plugin/Lumberjack.json. You can change the following params:

Basic options
Option Description
lumberjack_tool The item required to chop.
reward the object that is given at the end of felling
reward_item_count the count of reward given
chop_time The time it takes to cut down the tree
tree_respawn_delay The time it takes to regenerate the tree
Plugin messages
Message Description
need_axe The item required to chop.
equip_axe the object that is given at the end of felling
reward_message
